# 🌾 RozgarPath

> A CLI tool to help fresh graduates discover practical and profitable career paths — especially for people in agriculture-based regions.

---

## 📌 Overview  
RozgarPath is a command-line tool designed to guide graduates who have academic degrees but lack practical skills or direction. It provides structured career guidance, helping users move from confusion to clarity with actionable steps.

---

## ❓ Problem Statement  
Many graduates complete their degrees without gaining real-world skills. As a result, they face:

- ❌ Lack of clarity about career direction  
- ❌ No practical experience  
- ❌ Difficulty entering the job market  

RozgarPath solves this by offering step-by-step guidance tailored to the user’s background.

---

## ⚙️ What It Does  
- 👤 User enters their name and degree  
- 🧭 Shows 3 career paths matching their background  
- ✅ User selects a preferred career  
- 📚 Displays required skills for that path  
- 🌐 Provides free learning resources  
- 💾 Saves user progress for future use  

---

## 🎯 Features  
- 🧭 Career path suggestions based on user profile  
- 🌱 Focus on agriculture + business opportunities  
- 📋 Step-by-step actionable roadmap  
- 💡 Skill recommendations for each path  
- ⚡ Simple and fast CLI-based interaction  

---

## 👤 Target Users  
- 🎓 Fresh graduates  
- 🤔 Students unsure about their future  
- 🌾 Agriculture background students  
- 💼 Business graduates looking for practical paths  

---

## 🌍 Unique Value  
Unlike generic career tools, RozgarPath:

- 🔗 Combines **Agriculture + Business**  
- 🚀 Suggests **real-world, practical opportunities**  
- 💼 Encourages **entrepreneurship and self-employment**  

---

## 🛠️ Installation  

```bash
# Clone the repository
git clone https://github.com/your-username/rozgarpath.git

# Navigate into the project

```

## 🚀 How To Run  

```bash
python main.py
```

Follow the prompts in the terminal to explore your personalized career path.
